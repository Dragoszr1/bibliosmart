# BiblioSmart

O aplicație de gestionare a unei biblioteci folosind Flask,Mysql,vue,node.js

## Structura Site-ului

```
biblioteca/
├── backend/           # Flask API server
│   ├── app/
│   │   ├── routes/    # API endpoints
│   │   └── models/    # Database models
│   ├── config.py      # Configuration
│   ├── run.py         # Entry point
│   └── requirements.txt
├── frontend/          # Vue.js application
│   ├── src/
│   │   ├── components/  # Vue components
│   │   ├── pages/       # Page components
│   │   ├── App.vue      # Root component
│   │   └── main.js      # Entry point
│   ├── index.html
│   ├── vite.config.js
│   ├── tailwind.config.js
│   └── package.json
└── README.md
```
